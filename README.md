# camera_calib

这里提供一个gazebo中的仿真相机标定小工具。

测试了普通相机和鱼眼相机内参标定结果，较为准确。



## 运行环境

- `Platform` x86
- `System` Ubuntu 18.04
- `ROS` melodic
- `gazebo` 9.0.0
- `CMake` ≥ 3.0.2
